# Contributing

Thanks for helping improve Synapse Beats.

## Workflow

- Fork or create a feature branch.
- Keep PRs focused and small.
- Run local checks (`npm run dev`, `npm run build`) before opening PR.

## Standards

- Keep website branding as **Synapse Beats**.
- Do not add org/founder credits on public site UI.
- Keep credits in repository docs only.
